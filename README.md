This is my personalized Neovim setup based on [LazyVim](https://github.com/LazyVim/LazyVim), designed for an efficient and modern editing experience.

## Features

- Fast startup and lazy-loading of plugins
- Pre-configured LSP, Treesitter, and autocompletion
- Custom keymaps and UI enhancements
- Easily extensible and modular

## Getting Started

1. **Clone this repository:**
   ```sh
   git clone <your-repo-url> ~/.config/nvim
   ```

2. **Install required packages:**
   - `python3-venv`
   - `ripgrep`
   - `unzip`

   On Debian/Ubuntu:
   ```sh
   sudo apt install python3-venv ripgrep unzip
   ```

3. **Launch Neovim:**
   ```sh
   nvim
   ```

4. **Follow the [LazyVim installation guide](https://lazyvim.github.io/installation) for more details.**

## Customization

Feel free to explore and modify the configuration files in the `lua/` directory to suit your workflow.
